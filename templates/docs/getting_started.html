Getting Started

Find out how  pub2sum can be employed to automatically create human-like summaries for you.

  1. Navigate to the Summarizer page.

  2. Click on Choose file.

  3. Select a single PDF paper or a ZIP file containing multiple PDF papers from your machine and confirm the upload.

  4. Click on Summarize to start analyzing.

  5. Once done the site will refresh automatically showing you summaries for each paper including references and a reference list.

After that you can easily copy & paste the results to your own document.
